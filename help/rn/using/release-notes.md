## Release 26.1 (LA) {#26.1}

>[!AVAILABILITY]
>
>Deze release is maar voor een aantal organisaties beschikbaar (beperkte beschikbaarheid). Neem voor meer informatie contact op met uw Adobe-vertegenwoordiger.

### Beveiligingsoplossingen {#winter-26-security}

* Deze release biedt beveiligingsoplossingen.
* Deze release wordt geleverd met de volgende beveiligingsupdates: Debian 12 en PostgreSQL 17.

### Andere oplossingen {#winter-26-fixes}

* Verbeterde dynamische rapportage voor betere robuustheid en minder mismatches van gegevens. (CAMP-58964)
* Verbeterd IMS-aanmeldingsmechanisme voor betere betrouwbaarheid. (CAMP-59418)
* Verbeterde integratie van AEM-inhoud met betere URL-verwerking voor gepagineerde aanvragen. (CAMP-59540)
* Ondersteuning voor wachtwoordinvoer toegevoegd in het scherm Opties met automatische hashing. (CAMP-60206)
* Probleem verholpen waarbij tijdelijke werkstromen onjuiste reeks-id&#39;s gebruikten. (CAMP-59824)
* Probleem verholpen waarbij fouten konden optreden bij het ophalen van AEM-campagneinhoud. (CAMP-59926)
